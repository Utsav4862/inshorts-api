# Inshorts-api

This is an unofficial api for inshort news app

You can use it in your project also.

Its **FREE** to use

## How to use?
```
  https://inshort-api2.herokuapp.com/inShort?category={categoryName}
```
For Example: https://inshort-api2.herokuapp.com/inShort?category=all

## Categories

- all
- india
- business
- sports
- world
- politics
- technology
- startup
- entertainment
- miscellaneous
- hatke
- science
- automobile
